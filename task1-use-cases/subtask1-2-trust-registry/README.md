# Subtask 1.2: Trust Registry Use Cases

This subtask defines the use cases for operating the trust registry in the WP4 Trust Infrastructure.

## Folder contents

- **`README.md`** – Summary of trust registry use case areas in this subtask

## Covered trust registry operations

- **Trust establishment** – Initial trust setup, trusted list creation, trust anchor configuration, and trust chain validation.
- **Trust evaluation** – Certificate validation, policy compliance checking, trust level assessment and risk evaluation.
- **Trust maintenance** – Certificate renewal, policy updates, trust level adjustment, access control, approval workflows and delegation management.
- **Trust revocation** – Certificate revocation, trust suspension, emergency revocation and revocation list management.
- **Trust discovery** – Discovering trust registries, fetching data, resolving trust chains and receiving notifications.
