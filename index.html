WP4 — Trust Registry Infrastructure

List of Trusted Lists (LoTL)

Machine-readable List of Trusted Lists for the WE BUILD pilot, following ETSI TS 119 612 / TS 119 602. Use the endpoints below to fetch the signed LoTL in JSON or XML for trust validation in Wallet Units and Relying Parties.

Published endpoints

LoTL (JSON)

JAdES-signed JSON (TS 119 602-1 LoTE root, 1960201 shape).

Download JSON

https://webuild-consortium.github.io/wp4-trust-group/list_of_trusted_lists.json

LoTL (XML)

XAdES-signed XML (TS 119 612 ListOfTrustedLists).

Download XML

https://webuild-consortium.github.io/wp4-trust-group/list_of_trusted_lists.xml

Specification and contributor workflow (Trusted List Provider tl_entries pull requests, CI validation): LoTL automation and TL integration in the wp4-trust-group repository.
